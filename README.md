# 📊 BMI Tracker Pro

Aplikasi kalkulator kesehatan berbasis web menggunakan **Streamlit** untuk memantau Body Mass Index (BMI) dan mengukur estimasi berat badan ideal berdasarkan profil fisik pengguna.

---

## 🛠️ Persiapan Aset
Letakkan file gambar berikut pada direktori utama proyek bersama file `app.py`:
* `jarum.png` (Indikator visual grafik BMI)
* `gambar.jpg` (Latar belakang elemen header)

---
